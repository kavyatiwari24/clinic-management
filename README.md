# Aastha Clinic prototype

This is a complete React + Vite, Vercel-ready local-clinic management prototype. It uses fictional Indian patient data and Delhi addresses only.

## Run it locally

1. Install Node.js LTS.
2. Open this project folder in a terminal.
3. Run `npm install` once, then `npm run dev`.

## Deploy it on Vercel

1. Upload this entire folder to a new GitHub repository (do not upload only `src`).
2. In Vercel, choose **Add New → Project**, select the repository, and click **Deploy**.
3. Leave Vercel’s detected settings unchanged: framework **Vite**, build command `npm run build`, output directory `dist`.

The generated `*.vercel.app` link is public and has no Vercel watermark.
